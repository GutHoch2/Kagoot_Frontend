# Anleitung

## 1. Bootstrap installieren
`npm install bootstrap@5.3.3`

## 2. SAAS installieren 
`npm install sass`

## 3. Bootstrap einbinden in main.js
`import 'bootstrap';`

## 4. Bootstrap SCSS einbinden in main.js
`import 'bootstrap/scss/bootstrap.scss';`
